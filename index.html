MAFQOODAT

An AI-powered lost & found platform built at the RamadanIA Hackathon to digitalize and automate lost-item reporting and recovery. Features a conversational AI agent and AI-assisted matching workflows for individuals and institutions. Selected for the national final as a scalable centralized infrastructure.

02 MLOps · Research

Tropical Virus PhyloTree

Turns viral RNA/DNA sequences into an optimized phylogenetic tree by learning a tropical correction of the distance matrix, then running Neighbor-Joining. Full MLOps stack — React dashboard, FastAPI, Streamlit, Docker & CI.

  • Tropical Geometry
  • MLOps
  • FastAPI
  • React
  • Docker

03 Deep Learning

3D Point-Cloud Nets

Graph neural networks (ECC & MoNet) classifying 3D point clouds (CAP3D / ShapeNet), with Federated Learning (FedAvg IID / non-IID, VFL) and knowledge distillation compressing the model 4×.

  • GNN
  • Federated Learning
  • Distillation
  • PyTorch

Container / truck-loading optimizer for the 3D Bin Packing Problem — simplex, heuristics, metaheuristics and a reinforcement-learning approach for real warehouse logistics. Built at Smart Automation, Tangier.

  • Reinforcement Learning
  • Simplex
  • Python
